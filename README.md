## Introduction

This script is designed to simplify the task of converting strings to `camelCase` or `snake_case` and managing files within a user-specified "pages" folder. It is useful for developers who need to standardize string formats and interact with directory contents.

## Features

- **String Case Conversion**:

  - Convert strings from various formats (kebab-case, camelCase, PascalCase) to `camelCase`.
  - Convert strings from various formats to `snake_case`.

- **Pages Folder Management**:
  - Prompt the user to specify the path to a "pages" folder.
  - List files within the specified folder.
  - Display contents of a selected file from the folder.
